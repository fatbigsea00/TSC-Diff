# SSTar 数据集 (Side-Scan Sonar Targets)

SSTar 是本论文构建的侧扫声呐目标数据集。本仓库当前仅提供**少量样本实例**用于展示数据形式;**完整数据集及相关信息将在论文录用后公开**。

## 样本说明

`samples/` 下每个目标类别提供若干张原始侧扫声呐图像示例:

| 类别(原始编码) | 示例数量 |
|------------------|----------|
| RGYJ | 4 |
| SXJS | 4 |
| MTZ | 4 |
| JZX | 4 |
| shipwreck | 2 |

> 类别文件夹沿用数据集原始命名编码。此处样本仅作格式展示之用,完整的类别定义、标注与划分等信息将随完整数据集一并公开。
